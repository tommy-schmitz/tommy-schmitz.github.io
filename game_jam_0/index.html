About

You can control Mia using the w, a, s, and d keys. Left click with the mouse to cast a fireball spell next to Mia.

Your goal is to navigate the Perilous Swamps and make it out the other end alive.

Development

This game was made by Tommy and Eli as part of a game jam that went from Saturday, June 21 to Sunday, June 22, 2014. No external libraries or frameworks were used.

The background music was composed by Tommy using Noteworthy Composer. The sounds were created by Eli using Bfxr. Both Eli and Tommy used Audacity to finish the sound. The artwork was created with GIMP and Graphics Gale. The project was managed using Git through the BitBucket service.
