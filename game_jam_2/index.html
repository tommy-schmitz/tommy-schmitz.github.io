About

Move around using the w, s, a, d keys. Left-click on resources to gather them. Hit 'e' to open a crafting menu. Click on an item to craft it. Hit 1, 2, 3, 4 to select an item then left-click to place it. Hit the grave key (`/~) to enable delete mode, and left-click a tower to delete it.

Collect resources and build defensive towers to survive as long as possible against the zombie hordes!

Development

This game was made by Carolyn, Joe, Tommy, Shawn, and Eli as part of a game jam that went from Saturday, August 2 to Sunday, August 3, 2014. The basic framework was copied from our previous game jam projects, Mia the Mage and MonTact. Other than that everything in this game was made within those two days.

We used the following programs to create this game:

  • Music: Noteworthy Composer
  • Sound: Bfxr
  • Sound: Audacity
  • Art: GIMP
  • Art: Adobe Photoshop
  • Art: Adobe Flash
  • Version Control: Git
  • Code Hosting: BitBucket
