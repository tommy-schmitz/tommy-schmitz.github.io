Controls

Keyboard controls:
  • Spacebar: jump.
  • [F] key: shoot.
  • Arrow keys: move around.
Touchscreen controls:
  • Touch the screen to make the touchscreen controls appear on the screen.
  • A: jump.
  • B: fire.
  • D-pad: move around.

About

This game is inspired by Mega Man, made by Capcom (official website: link).

Development

This game was made by Joe, Shawn, Eli and Tommy in five days: Saturday, September 20 to Wednesday, September 24 (2014).

The basic framework was copied from our previous game jam projects. Images of Mega Man and Sniper Joe were taken (and partially adapted) from the original Mega Man games via various internet sources ("Google image search"). All other parts (other artwork, code, and music) were created by our team.
